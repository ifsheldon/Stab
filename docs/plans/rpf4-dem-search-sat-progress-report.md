# RPF4 DEM Search And SAT Progress Report

## Scope

This report records the RPF4 progress slice for current capped-repeat graphlike search, hypergraph search, SAT problem generation, analyzer traversal, and ErrorMatcher traversal.
It also records the later zero-probability repeat-skip slices for graphlike search and hypergraph search, selected flat detector-touching and detectorless logical-only zero-shift repeat folding for graphlike and hypergraph search, zero-probability variable elision and repeated-body skipping for weighted SAT/WCNF generation, the selected flat zero-shift SAT/WCNF repeat-folding slice, and the selected unweighted zero-probability structural SAT repeat-folding slice.
It also records the selected flat detector-touching zero-shift ErrorMatcher filter DEM repeat-folding slice.

This is not a full folded-traversal implementation. Graphlike and hypergraph traversal skip repeated bodies that contain no nonzero-probability error mechanisms and fold selected flat detector-touching or detectorless logical-only zero-shift repeated error bodies into compact search-graph edges, weighted SAT/WCNF omits zero-probability error variables and skips repeated zero-probability bodies before flattening, weighted SAT/WCNF folds selected flat zero-shift repeat bodies by concrete MAP parity cost for nonzero mechanisms, unweighted SAT folds selected flat zero-shift repeat bodies structurally including zero-probability mechanisms, and ErrorMatcher filter DEM traversal folds selected flat detector-touching zero-shift repeated filter keys. Shifted, nested, non-flat, numeric-target, detectorless no-target, analyzer traversal, ErrorMatcher circuit traversal and provenance, and high-index dense-target structural SAT repeats still keep explicit caps or current dense target limits.
