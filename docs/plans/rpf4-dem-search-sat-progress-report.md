# RPF4 DEM Search And SAT Progress Report

## Scope

This report records the RPF4 progress slice for current capped-repeat graphlike search, hypergraph search, SAT problem generation, analyzer traversal, and ErrorMatcher traversal.
It also records the later zero-probability repeat-skip slices for graphlike search and hypergraph search, selected flat detector-touching and detectorless logical-only zero-shift repeat folding for graphlike and hypergraph search, selected flat no-target zero-shift repeat skipping for graphlike and hypergraph search, selected flat zero-detector-shift `shift_detectors 0` repeat folding for graphlike and hypergraph search, selected flat annotation-bearing repeat folding for graphlike and hypergraph search, selected nested zero-shift repeat folding for graphlike and hypergraph search, selected mixed zero-probability plus active zero-shift repeat folding for graphlike and hypergraph search, zero-probability variable elision and repeated-body skipping for weighted SAT/WCNF generation, the selected flat and nested zero-shift SAT/WCNF repeat-folding slices, and the selected unweighted zero-probability structural SAT repeat-folding slice.
It also records the selected flat and nested detector-touching zero-shift ErrorMatcher filter DEM repeat-folding slices.

This is not a full folded-traversal implementation. Graphlike and hypergraph traversal skip repeated bodies that contain no nonzero-probability error mechanisms, fold selected flat detector-touching or detectorless logical-only zero-shift repeated error bodies into compact search-graph edges, skip selected flat no-target zero-shift repeated error bodies, fold selected flat `shift_detectors 0` repeated bodies, fold selected flat annotation-bearing repeated bodies with `detector` or standalone `logical_observable` declarations, fold selected nested zero-shift repeated bodies, fold selected mixed zero-probability plus active zero-shift repeated bodies while ignoring zero-probability search targets, and now use sparse internal detector-node indexes for selected shifted zero-probability bodies that place later active graphlike or hypergraph errors at high but sparse detector ids, weighted SAT/WCNF omits zero-probability error variables and skips repeated zero-probability bodies before flattening, weighted SAT/WCNF folds selected flat and nested zero-shift repeat bodies by concrete MAP parity cost for nonzero mechanisms, unweighted SAT folds selected flat and nested zero-shift repeat bodies structurally including zero-probability mechanisms, and ErrorMatcher filter DEM traversal folds selected flat and nested detector-touching plus detectorless logical-observable-only zero-shift repeated filter keys. Nonzero-shift active repeats, shifted nested active repeats, non-flat active repeats, broader non-annotation mixed-instruction search repeats, analyzer traversal, ErrorMatcher circuit traversal and provenance, annotation-only or broader mixed-instruction filter bodies, and high-index dense-target structural SAT repeats still keep explicit caps or current dense target limits. Raw numeric `error` targets and separator-only `error` target lists are rejected before search traversal and are locked by [pfm4-dem-search-invalid-target-boundary-scope.md](pfm4-dem-search-invalid-target-boundary-scope.md).
